# Content Frameworks & Systems

Repeatable systems for producing content at scale without burning out.

## Core Principles

- **Batch production** — Film many, edit later
- **Content repurposing** — One piece → 10 formats
- **Systemize creativity** — Templates free up mental energy
- **Track what works** — Data-driven decisions
- **Build a content bank** — Never start from zero

## By Creator

### [[Alex Hormozi]] — The Volume System
- 50 hooks × 3-5 meats × 1-3 CTAs = **150-750 ads per week**
- Film all hooks, meats, CTAs in one session
- Edit together for maximum volume

> "Instead of creating one ad, I write 50 hooks." — Hormozi

**Source:** [[Alex Hormozi — Ad Volume Framework]]

### [[Gary Vaynerchuk]] — Document, Don't Create
- Film your real life, not scripted content
- 60+ pieces of content per day from one keynote
- "Always be macro-dosing on content"

### [[Ali Abdaal]] — The Content Pipeline
- Script → Film → Edit → Distribute
- Batch each stage
- Use templates for repeatable formats

## Content Repurposing Framework

### One Piece → Many Formats
```
YouTube Video
├── 3-5 Shorts/Reels (key moments)
├── Blog post (transcript)
├── Twitter thread (key points)
├── LinkedIn post (professional angle)
├── Instagram carousel (visual summary)
├── Email newsletter (deep dive)
└── Podcast audio (extract audio)
```

## Batching System

### Weekly Batch Workflow
- **Monday:** Plan + script 5-7 pieces
- **Tuesday:** Film all videos
- **Wednesday:** Edit batch
- **Thursday:** Write copy + captions
- **Friday:** Schedule + analytics review

### Monthly Content Calendar
- Week 1: Educational content
- Week 2: Story/narrative content
- Week 3: Trend/reactive content
- Week 4: Promotional/CTA content

## Quality Control Checklist

- [ ] Hook grabs attention in first second
- [ ] Clear value proposition
- [ ] One CTA per piece
- [ ] Subtitles/captions included
- [ ] Thumbnail optimized (if YouTube)
- [ ] Posted at optimal time
