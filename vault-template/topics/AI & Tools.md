# AI & Tools

AI workflows, automation, and tools for content creation and marketing.

## Core Principles

- **AI augments, doesn't replace** — Your voice still matters
- **Prompt engineering = skill** — Better prompts = better output
- **Automate the boring stuff** — Use AI for research, editing, repurposing
- **Stay current** — Tools change fast, frameworks last

## AI Content Workflows

### Repurposing Pipeline
```
1 Video → Transcript (Whisper) → Blog post (Claude) → Social threads (Claude) → Newsletter (Claude)
```

### Research Pipeline
```
Topic → Web research (Perplexity) → Key insights (Claude) → Script (Claude) → Video
```

### Editing Pipeline
```
Raw footage → Auto-cut (Descript) → Captions (CapCut) → B-roll suggestions (AI) → Final edit
```
