# Marketing & Strategy

Positioning, funnels, brand building, and growth strategy.

## Core Principles

- **Positioning is everything** — How you're perceived determines what you can charge
- **One audience, one message** — Don't try to be everything to everyone
- **Brand = perception** — Every touchpoint shapes it
- **Funnels are just journeys** — Map the customer journey, then optimize

## By Creator

### [[Alex Hormozi]] — Volume + Variety
- "Variety is the key to volume"
- Test everything, keep what works
- 150-750 ads per week = statistical advantage

### [[Iman Gadzhi]] — One-Person Media Company
- Own the distribution
- Build audience before product
- Education as marketing

## Frameworks

### The Marketing Stack
```
Awareness → Interest → Desire → Action → Retention → Referral
```

### Hormozi's Value Equation
```
Value = (Dream Outcome × Perceived Likelihood) / (Time × Effort)
```

### The AARRR Funnel
```
Acquisition → Activation → Retention → Revenue → Referral
```

## Brand Building

### Brand Pillars
1. **Mission** — Why you exist
2. **Values** — What you stand for
3. **Voice** — How you communicate
4. **Visual** — How you look
5. **Story** — Your origin narrative

### Content Pillars (pick 3-5)
- Educational
- Entertaining
- Inspirational
- Promotional
- Behind-the-scenes

## Funnels

### Cold Traffic Funnel
```
Ad → Landing page → Lead magnet → Welcome sequence → Offer
```

### Warm Traffic Funnel
```
Content → Email list → Nurture sequence → Soft pitch → Sale
```

### Retention Funnel
```
Purchase → Onboarding → Success → Upsell → Referral
```
