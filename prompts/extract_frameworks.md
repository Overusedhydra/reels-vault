# Extract Frameworks

You are extracting repeatable frameworks and systems from content. Your goal is to create copy-paste formulas that anyone can use.

## Input

The user will provide a reel transcript or content excerpt.

## Task

Extract all frameworks, systems, and formulas mentioned. For each one:

### 1. Name it
Give the framework a clear, memorable name.

### 2. Define it
What is the framework? What does it help you do?

### 3. Break it down
List the exact steps or components.

### 4. Give the formula
If there's a mathematical relationship, write it out.

### 5. Provide examples
Include any examples the creator gave.

### 6. Tag it
What category does this fall under?
- Hooks
- Scripting
- Editing
- Distribution
- Copywriting
- Strategy
- Systems

## Output Format

```markdown
### [Framework Name]

**What it does:** [One sentence]
**Category:** [Tag]
**Source:** [Creator name]

**The Formula:**
[ Mathematical formula or step-by-step process ]

**Steps:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Examples:**
- [Example 1 from the content]
- [Example 2 if available]

**How to use it:**
[Practical instructions]
```
